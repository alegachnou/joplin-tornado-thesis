# joplin-tornado-thesis

This repository is a research package for the 2011 Joplin tornado thesis. It includes analyses, data collection, and findings related to the tornado event.

## Features
- Detailed data analysis of the Joplin tornado
- Visualizations of the data

## Technical Stack
- Python for data analysis
- Matplotlib for visualizations

## Usage
Clone the repository and run the analysis scripts to view the findings.

## Future Plans
- Add more visualizations
